# webtoonBot

## **Summary**
해당 프로젝트는 웹툰을 너무 좋아하는 제가 필요해서 만든 WebtoonBot입니다. 
제가 설정해놓은 시간대에 제가 보는 웹툰 목록을 확인해서 업로드 된 것들만 저에게 메일로 알려주는 기능을 합니다.
WebtoonBot을 만들기 위해 아래와 같은 과정을 계획하였습니다.

## **Step**

1. 먼저 윈도우에서 해당 해당 프로젝트 구현한다.

```
1-1. 현재 요일을 가져온다.
1-2. 해당 요일에 따라서 해당요일의 네이버 웹툰 페이지url로 이동한다.
1-3. 내가 보는 웹툰 목록에서 up이 뜬 목록만 가져온다.
1-4. 그 목록을 내 gmail로 발송한다.
```

2. 해당 프로젝트를 git을 이용해서 서버로 pull한다.

3. 서버에서 크론탭을 이용해서 매일 23:30, 00:00, 00:30에 해당
   프로젝트를 실행한다.
